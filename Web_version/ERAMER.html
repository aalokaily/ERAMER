ERAMER: A Novel Tool for Predicting the Trimming by ERAP1 Enzyme
The tool predictes a score for trimming a given peptide/s by ERAP1 enzyme.



Input peptides in fasta format (an example of fasta format is at fasta format example ), or
upload a fasta file
